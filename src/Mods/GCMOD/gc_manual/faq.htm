TECHNICAL SUPPORT
Common Questions

FAQ:

Common Questions:

Q: What is Galactic Conquest? 

A: Galactic Conquest is a team based first person shooter, based in the Star Wars Universe, built on the Battlefield1942 engine. It is a total conversion, with all new art, vehicles, gameplay, and features.

-------------------------------------------------------------------------------------

Q: Will there be Jedi? Can I suggest Jedi? I know how you could do Jedi.

A: Galactic Conquest is designed to feature the grunt warfare and epic space battles of the original Star Wars timeline. Jedi do not fit into this theme, and are not part of our design plans. We know we could do them, we just don't want to at this point.

-------------------------------------------------------------------------------------

Q: When will Galactic Conquest have it's next update?

A: Galactic Conquest is constantly growing and evolving, but as it is a free time modification, it is very hard to set long term release dates. As such, the general response is "when it's ready." However, if you frequent our forums at www.galactic-conquest.net, we usually update on our status, and that can give you a rough idea.

-------------------------------------------------------------------------------------

Q: There are no parachutes?

A: This is a feature of Galactic Conquest. Transport pilots are required to LAND, in order to let off troops. GC transports are tough vehicles tough, and can do a decent job of landing under fire.

-------------------------------------------------------------------------------------

Q: I don't like something, can it be changed?

A: GC is always changing, and being refined. We have a forum setup specifically for suggestions from players, and we watch the community closely to see how they enjoy the game. Check out our website at www.galactic-conquest.net, and make any suggestions you like. Though we ask that you stay civil about it. ;-)

-------------------------------------------------------------------------------------

Q: Will there be single player bot support for GC? 

A: We are not planning single player support directly, GC is a multiplayer-only project. Many of our map setups are just to complicated for the BF AI units. However, there is a mini-mod of GC fans are creating, called GCX, that is working on adding in single player maps, and adjust those that will work. Check out the GC forums for details.

-------------------------------------------------------------------------------------

Q: Will there be ATAT's?

A: The player fascination with the ATAT, over the 50 other vehicles we have in the mod, is quite shocking. ATAT's are very hard to make with the BF engine, which doesn't support vehicle animation states needed to make the walking legs believable. Long story short: We're working on it constantly, and trying new tricks, but there is no solid ETA for when we feel we'll have it looking good. We'd love to have it in game, but we want to make sure we do it right.

-------------------------------------------------------------------------------------





TECHNICAL SUPPORT Q&A:

Q: When I run GC, the menu is changed, but the maps are all still Battlefield ones, and I don't see new vehicles!?

A: Galactic Conquest is multiplayer only, and is not supported in single player battlefield mode, nor does it have bot support. You'll need to play online, or over a LAN network. If you just want to try out the vehicles and maps by yourself, you can do so by creating your own LAN game. Also, you need to make sure you load in the GC maps (Maps with GC_Name as titles) and don't create a game with normal BF maps.

-------------------------------------------------------------------------------------

Q: When I run GC, the vote menu screen, and scoreboard pops up, and I can't get it to go away!

A: You need to update to the latest version of Battlefield1942, from battlefield1942.com. GC requires the most recent version of the game to run correctly.

-------------------------------------------------------------------------------------

Q: I think the instillation didn't work?

A: You need to make sure when you run the installer you specify the directory to which you made your Battlefield1942 install. The default path may not be correct, if you installed or changed the BF directory name.

-------------------------------------------------------------------------------------

Q: I join a game, but I don't have any textures or sounds! But I can see grey models.

A: You must have downloaded the SERVER ONLY version of BF1942, which was for server hosts that run deticated servers, and don't need the art files. You need to go back and download the CLIENT version of GC, which will be the one that is about 400megabytes.

-------------------------------------------------------------------------------------

Q: When I join a server online, the game crashes half way through the map load process!

A: You are running a different version of Galactic Conquest than the server is. Check our website to make sure you have the most up-to-date version of GC. Also, it's possible the Server Admin is still running an older version, in which case you can only wait for that server admin to update.

-------------------------------------------------------------------------------------

Q: Where can I get more help? 

A: The technical support forums at www.galactic-conquest.net.
